# 🚀 @obukata/project-starter

社内で小規模Webサイトをすばやく立ち上げるためのスターターテンプレート集です。  
Tailwind CSS、PostCSS、Vite など、共通で使用する開発環境を自動セットアップします。

---

## 📦 インストール方法

プロジェクトを新規作成するには、以下のコマンドを実行します：

```bash
npx @obukata/project-starter
```

実行すると、ターミナル上でテンプレートの種類を選択し、
プロジェクト名を入力するだけで、環境構築が自動で完了します。

---

## 🧩 テンプレート一覧

| テンプレート名      | 説明                          | npm install の実行場所 |
| ------------ | --------------------------- | ----------------- |
| **MJサーバ用**   | 自社サーバ環境向けテンプレート（public配下構成） | `./public/`       |
| **制作パートナー用** | 外部パートナー向けテンプレート（標準構成）       | `./`              |

---

## ⚙️ 使用方法

1. テンプレートを選択

コマンド実行後に、以下のような選択肢が表示されます：

```bash
どのテンプレートを使用しますか？
> MJサーバ用
  制作パートナー用
```

2. プロジェクト名を入力

```bash
プロジェクト名を入力してください: my-site
```

3. 自動セットアップ

テンプレートがコピーされ、npm install が自動で実行されます。
完了後は、次のようなメッセージが表示されます：

```bash
✅ テンプレートコピー完了
📥 npm install を実行中...（my-site/public）
🎉 完了! "my-site" フォルダが作成されました。
```

---

## 🪄 ローカル開発

テンプレートセットアップ後は以下のコマンドで開発サーバを起動できます：

```bash
npm run dev
```

（パートナー用）ビルドする場合

```bash
npm run build
```
